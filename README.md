Basándonos en el aprendizaje del curso de programación, realizamos un proyecto de Backend,  una API REST construida con Flask (Python) y que utiliza MySQL como base de datos, y por el lado del Frontend como aplicación de una pagina web, de tienda de componentes construida con React, con diseño moderno usando utiliza Tailwind CSS para su diseño visual, y que utiliza Firebase (Auth y Firestore) para la autenticación y persistencia de datos en el lado del cliente.
